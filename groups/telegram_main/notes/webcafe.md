## 我现在知道的（持续更新）

### 一、核心框架：出海独立开发者赚钱路径

> **"上站，上站，想要赚美元就多上站"** —— 哥飞

路径：**挖新词找需求 → 快速做站 → SEO搞流量 → Adsense/支付变现**

### 二、SEO核心公式

> **SEO = 内容(80%) + 外链(20%) + 技术(0%)** — 子木

### 三、出海飞轮（核心循环）⭐

> **"会做一个页面，就会做多个页面；会做一个语言，就会做更多语言；会做一个网站，就会做多个网站。"** —— 哥飞

**飞轮执行SOP（四阶段）**：
```
单页（会做1个能拿SEO流量的页面）
    ↓ 复制
多页（围绕核心关键词扩展更多相关页面）
    ↓ 扩展
多语言（用子目录做各语言版本，获取全球搜索流量）
    ↓ 复制
多站（换一个关键词/语言，重复上述过程）
```

**关键点**：
- 核心能力是"做一个能拿SEO流量的单页"，这个会了后面都是复制
- 用子目录做多语言，冷启动成本最低（不用同时冷启动多个域名）
- 多语言要找当地语言的用户会搜的词，不是直接AI翻译

**多语言SEO三大铁律**：
1. 不用JS切换语言，每种语言必须有独立URL（子目录 `/en/`）
2. 不用AI直接翻译，要用当地语言工具找用户实际搜索的词
3. hreflang标签告诉谷歌各语言版本的对应关系，必须正确配置

**pokerogue案例（多语言威力证明）**：
- pokerogue.net / pokerogue.io 三个域名都没做多语言
- 群友做了多语言，3个月：244万曝光，45万点击，外链只有26个
- 说明多语言是流量杠杆，外链需求被大幅降低

### 四、需求挖掘方法论

#### 找热点新词（Trending Keywords）

- 新词定义：出现时间不超过6个月的词，搜索量急剧上升但大站尚未完全覆盖
- 避开竞争：从搜索量上升但已有页面不多的词切入

#### 找新词的科技媒体渠道（按优先级）

1. **Twitter/X** — AI公司官方账号（OpenAI等）
2. **GitHub Trending** — github.com/trending（发现开源工具名称）
3. **Huggingface 趋势榜单** — huggingface.co/models?sort=trending（发现AI模型名称）
4. **Product Hunt** — 新产品发布平台
5. **微博热搜 / 知乎热榜 / 抖音热点** — 中文热点
6. **Vercel 子域名** — 冷门工具入口
7. **Gumroad 高流量页面** — 别人正在赚钱的需求

#### KDRoi公式（关键词性价比公式）⭐

```
KDRoi = (搜索量 × 点击单价) / 优化难度
```

- 用 Semrush 导出关键词数据计算
- KDRoi 越高，说明这个词越值得做
- **核心洞见**：离钱越近的关键词越值钱。金融/法律/B2B类CPC高但竞争大；Generator/Converter/Calculator类工具词是新手最佳切入点——CPC中等、KD可控。
- 搜索量大的词不等于值得做的词

**KDRoi对比示例**（袁锐钦）：
- AI Headshot Generator：搜索量8100，CPC $3.5，KD 18 → KDRoi = 1575 ✅
- Password Generator：搜索量110000，CPC $1.2，KD 92 → KDRoi = 1433
- Invoice Generator：搜索量22000，CPC $4.2，KD 45 → KDRoi = 2053 ✅
- QR Code Generator：搜索量45000，CPC $0.8，KD 72 → KDRoi = 500

### 五、SEO方法论

#### 两大原则

1. **分门别类罗列** — 清晰的信息架构
2. **一个关键词一个页面** — 谷歌排名的最小颗粒度是页面，不要让多个页面竞争同一个词

#### 关键词蚕食（Keyword Cannibalization）

- 同一网站多个页面竞争同一关键词 → 权重分散
- 解法：举全站之力优化一个关键词，首页做核心词，内页做长尾词

#### On Page SEO 清单

- **TDK**：标题含关键词（占排名权重约15%）、描述150-160字符
- **H1-H6**：每个页面一个H1，H2/H3辅助层次清晰
- **图片alt**：必须写，利于Google图片搜索
- **canonical标签**：指定规范URL，集中权重
- **Schema Markup**：结构化数据标记
- **后端渲染**：避免纯Vue/React SPA，不利于SEO

#### 内容策略：农村包围城市

- 从 long-tail（长尾词）对应的1000个页面入手
- 初期权重低 → 先拿小词排名 → 积累外链 → 权重提升 → 拿大词排名
- 瞄准大词A但先拿到小词AB/AC流量是**正常路径**，权重提升后才竞争大词

#### 海量页面（Programmatic SEO）

- 用程序化方式生成大量页面覆盖关键词
- 适合：有明确数据源的场景（如某类工具、游戏名称列表）
- ⚠️ 风险警告：批量上低质量页面会被谷歌判定为垃圾页面并K站

#### Information Gain（信息增量）⭐

- 谷歌2020年申请专利，描述基于机器学习的智能网页推荐系统
- 核心思想：你的内容必须比现有结果**提供额外价值**，才能获得好排名
- 为什么要做新词新站？新词没有现成好内容 → 信息增量容易做到 → 快速起量
- 反面教训：Woy.ai因为批量上低质量页面被K站

#### 收录机制（永无bug）⭐

1. **爬取 → 评估内容质量 → 决定是否收录**
   - 低质量内容特征：内容空洞/重复/模板化/疑似AI批量生成
   - 结果："已抓取 - 尚未编入索引"
   - 验证方法：GSC网址检查 → 请求编入索引 → 一直失败 = 质量问题

2. **高权重域名可以弥补内容质量的不足**
   - 大站内页内容质量也不高，但老域名+大量外链 → 谷歌信任 → 直接收录
   - 本质公式：**是否收录 ≈ 网站权重分数 + 网页内容质量分数**

3. **排名是动态的，用户行为决定最终排名**
   - 新页面初始排名由域名权重决定（长尾词通常给不错初始排名）
   - 4-5小时后：谷歌观察用户行为 → 动态调整
   - 用户进入后马上跳走 → 排名下降
   - 用户停留时间长 → 排名保留或上升
   - **跳出率是内容质量的最直接信号**

4. **留住用户不能只靠文本内容**
   - 文本是底线（决定收录），但留住用户靠：图片/视频（吸引用户停留）+ 可交互内容 + 内链引导

#### 内容AI生成原则（哥飞养网站防老第9步）⭐

核心原则：**不要让AI"凭空捏造"**

1. **提供参考资料**（链接/数据源），让AI基于真实来源创作 → 避免幻觉，内容有据可查
2. **指定输出格式**（半结构化Markdown）→ 统一风格，更容易被搜索引擎识别为高质量

### 六、外链建设方法论

#### 外链建设5种方法

1. **链接诱饵（Link Baiting）**：制作行业报告、免费工具（计算器、分析器），别人主动引用，无需主动要

2. **客座博客（Guest Posting）**：找高权重相关博客投稿
   - 找KOL技巧：用头像图片反搜他在哪些网站发过文章，针对这些站点投稿
   - 找法：Semrush/Ahrefs输入竞品域名 → Backlinks → 找Guest Post类型的链接

3. **HARO平台**：记者发布采访需求 → 你提供专业见解 → 获得权威媒体链接

4. **失效链接修复（Broken Link Building）**：Ahrefs/Semrush找竞品/行业博客404死链 → 联系站长建议用你的内容替代

5. **快速收录渠道**（新站特有）：V2EX / 阮一峰周刊 / 小众软件 / Product Hunt
   - 新网站被收录速度 ≈ 谷歌蜘蛛到达你提交链接网站的频率

#### 博客评论外链（Blog Comment Backlinks）⭐ 重要

**原理**：早期博客程序默认无需登录注册即可评论，评论者填写的个人网站URL会显示在昵称旁，形成外链

**为什么有效**：全球可能还有几十万到几百万个这样的老博客站；小游戏站竞争不激烈，低权重博客评论外链靠数量弥补足够拿到排名

**适用场景**：小游戏站（竞争弱）；AI工具站可作为**辅助外链**增加外链域名总数，震慑竞争对手

**不适用**：竞争激烈的AI视频站等，nofollow外链比例高，需要高质量外链才能竞争

**自动化发现飞轮**（哥飞演示）：
```
小游戏站外链列表 → 找博客评论外链的站 → 查这些站的外链
    ↑                                              ↓
    ←———— 更多博客评论文章 ←—————————————
```
哥飞实测：一上午自动发现841个外链，筛选出226个无需注册的可发布资源

**AI辅助发布（Gemini 2.0 Flash）**：
- 自动生成评论内容 + 填写所有输入框，用户只需点击提交
- 半自动：AI填写 → 用户审核 → 提交
- 全自动：直接发布（需信任AI）
- 成本：2K请求 + 10M Token = **$1.13**，每个外链约 **$0.00006**

**nofollow外链也有权重提升作用**（哥飞确认）——别嫌弃nofollow

#### ⚠️ demotion_signal警示

forum外链比例过高会触发谷歌降权信号（大罗提醒）

#### 落地页独立原则（HJ & Gemini 2026-02-04）⭐

- **核心结论**：每个落地页必须独立做外链，不用担心同域名重复
- **原理**：同域名多外链权重边际递减，但高DR站点"第二票"仍远超低质新域名"第一票"（DR80的20%>>DR10的100%）
- **关键操作**：在同高DR域名**发新文章**链向不同落地页，不能在老文章里硬加链接；锚文本要匹配各页面核心关键词；尽量只链向目标页面
- **深层链接价值**：提升整个网站可信度结构；Google喜欢自然链接结构（所有外链只指向首页不自然）

#### 高质量外链5条定义（哥飞）⭐

1. DR不低
2. 有自然搜索流量（来自搜索引擎）
3. 在某些关键词下能拿到搜索引擎靠前排名
4. 出站域名少（链接不要卖给太多人）
5. 入站域名多

#### 外链核心原则

- 相关性 > 数量
- Dofollow传权重，锚文本混合品牌词+通用词
- **外链域名数>总链接数**：买100个$30外链 > 买10个$300外链，付费外链也看域名数量
- **外链语言影响**：中文外链帮中文站排名，英文外链帮英文站排名

#### 高质量外链收集方法（安于不安）⭐

**核心**：在日常找词的同时**顺便**收集外链，流程并行，效率翻倍

**旧流程**：找词 → 上站 → 找同行 → 收集外链 → 发外链（串行）

**新流程**：找词(+同时收集外链) → 上站 → 筛选高质量外链 → 发外链（并行）

### 九、KGR vs KD：两个竞争维度指标（哥飞小课堂）⭐

**KGR（Keyword Golden Ratio）** — 聚焦**网页供应量**
- 核心看 `intitle:关键词` 的结果数量
- intitle大 → 说明大量网页已在标题里放了这个词 → 竞争者数量多
- KGR高 = 供应充足，通常出现在**老词**

**KD（Keyword Difficulty）** — 聚焦**竞争者质量**
- 核心看**前20名结果的权重**：外链数量、DR/DA、是首页还是内页
- Ahrefs看前10名外链数量；Semrush综合考虑SERP特征+外链+域名权重
- KD高 = 前面的页面强，不一定intitle数量多

**两个指标不一定正相关！**
- KGR低 + KD高：intitle少（前10页里大站内页多）= 机会
- KGR高 + KD低：网页多但竞争者弱
- 两者必须**同时看**，从不同维度判断

### 十、KD与所需外链数量公式（回车推导）⭐

> Ahrefs的KD对照表 → 数据拟合 → Michaelis-Menten公式

```
y = (-89.32212 × x) / (-100.6193 + x)
```

其中 x = KD值，y = 所需外链数量（估算）

| KD  | 所需外链（估算） |
| --- | ---------------- |
| 5   | ~5               |
| 10  | ~9               |
| 20  | ~20              |
| 30  | ~33              |
| 40  | ~47              |
| 50  | ~63              |
| 60  | ~74              |
| 70  | ~85              |
| 80  | ~95              |

> 注意：这是**估算值**，仅供参考。实际还需要考虑外链质量、域名权重、是否首页竞争等因素。

### 十一、高KD词策略（aifaceswap.io案例）⭐

**案例核心数据**：
- 域名：aifaceswap.io，2024年2月注册（5个月新站）
- 关键词：face swap ai，日搜索量2-3万，KD约70（需要~85个外链）
- 结果：只用74个外链 → **进入前5名**（超越了有1000+外链的竞品）
- 评论区后续追踪：目前1200万流量，持续增长

**为什么74个外链能赢1000+外链的竞品？**
- 完全免费使用，无需登录
- 用户体验更好（停留时长更长）
- 关键词密度更高
- **外链只是入场券**，决定排名的是**用户是否喜欢你的网站**

**高KD词（KD 50-70）打法**：
- 搜索量10万以内的词，机会很多，不需要做到第一，吃到一点就很赚
- 不要被KD吓跑，**外链是入场券，不是胜负手**
- 举全站之力用**首页**打别人的内页 → 以弱胜强
- 关键：产品更好用、更免费、效果更好

**新人最佳切入**：新词(<6月) + 前10名多为大站内页 + CPC尚可 + Exact Match Domain全押

**双轨策略**：新词靠运气可遇不可求，老词（KGR低的）长期稳定。**两条腿走路：新词遇到了就做，长期挑战老词**

### 十五、GSC信号解读（谷歌搜索控制台）⭐

**哥飞核心四句话（内页外链判断标准）**：

1. 任何链接都能传递权重（内链=外链）
2. 谷歌最小排名单位是页面
3. 任何页面拿到排名 = 竞争力足够
4. 任何关键词搜索结果 = 排行榜

**判断某页面能否拿排名的5步检查**：

1. On Page SEO 做好了？（主题聚焦）
2. 竞争力够？（外链权重+用户行为数据）
3. 域名权重够？（DA/DR）
4. 内容信息增量够？（比现有结果好在哪）
5. 技术SEO没问题？（canonical/hreflang/sitemap）

**新站被谷歌标记为危险网站的应对**：及时在GSC申诉，1-2天内快速处理

### 十六、从收录到出词完整SOP（哥飞小课堂2025-01-20）⭐

前提：做好了关键词研究，围绕关键词做好功能，每个页面围绕核心关键词写好内容

**第一步：提交GSC，等待收录**
- 内容没问题的话，24小时内可被收录
- 想加速：在爬虫频繁的外链站提交，或在自己有流量的网站放外链
- 合适外链刺激下，最快**2小时被收录**

**第二步：出词（收录后24-48小时）**
- GSC效果页面看到曝光了，说明出了词
- 48小时仍无动静 → 文案问题：内容长度不够/关键词分布不合理/TDH没覆盖/密度不够/前端渲染爬虫没抓到
- 出了词但不是想要的词 → 文案有问题（UGC内容不要误入落地页，防止稀释关键词密度）

**第三步：加外链，提升排名**
- 一开始排名可能在80-100名，没关系，加外链就能上升
- **最好方法：抄作业** → 谷歌搜索相关关键词 → 用Ahrefs/Semrush导出竞品外链 → 多找几个竞品取并集 → 越多网站有的外链越先去加
- 那么多网站能拿到，说明这种外链容易获取
- 零权重新站：**质量再差的外链也比没有好**，先拿下第一批外链才是最重要的
- 竞争不激烈的关键词（新词），加几个外链后效果立竿见影
- **外链有没有用，取决于你当前权重**：权重越低，低质量外链也有效；权重越高，越需要高质量外链

**第四步：持续维护**
- 评判外链是否有用，要看你当前权重
- 越到后面，网站权重越高，想提升DR需要的外链质量越高

### 十七、田七找词+稳定排名完整复盘（2026-02-15）⭐

**找词方法升级**：
- 新手：直接去Trends里找词，每天查；30天内看大趋势→记下大词→7天内看飙升关联词→2-3天custom日期找最新词
- **相关主题**（Related Topics）：一次只查一个词，看top主题最近在涨什么；查到原因后追源头（如roblox→xxx codes→新游戏工具）
- **Trending Now板块**：查印尼和巴西两国趋势词（两国互联网人口多，火了几天就必定会火；美国查出的词很多昙花一现）
- 用国家地区交叉验证：某词只有某国搜→去Reddit/X社区看是否只有该国人在讨论，是否可能全球流行
- **第一手信息永远在社媒和社区**：对什么感兴趣就去哪刷，如开源大模型→Reddit llm频道

**上站节奏**：
- 刚注册域名就一定要上站，上了才有可能有流量
- 刚开始只要首页+一两个内页，最快速度
- 喂AI资料→AI出SEO文案+结构+内链结构→积累的模板上站
- 提交GSC
- 为了尽早收录：可以在抓取频繁的外链站提交，或在自己有流量的网站放一个外链

**冲刺一周（最关键）**：
- 谷歌会给大约一周的曝光期
- 每天加几个外链，每天加一些内页，每个内页有内链指向首页或想拿排名的页面
- 这一周排名会非常不稳定（除非竞争少），但不能懈怠
- 不稳定时千万不要放弃，第二三周才会渐渐平稳（前提是站真正能满足了需求）
- **勤劳是唯一能让你坐稳位置的**（竞争维度差不多的情况下）
- 第三周后可减少外链频率，但仍不能完全断
- **初期不需要很多外链数量**，有一两个质量高一点的就能很快看到排名效果

**核心认知**：新词如果做不了最早一批，就不选择做；精力有限时要促使自己做更有把握的事

**Session 15（2026-04-02）**

**外链收集完整SOP**（安于不安 2025-03-23，收藏19）

核心方法：抄同行+顺手收集，在日常找词的同时顺便收集外链，不额外花时间。

原流程：找词 → 上站 → 找同行 → 收集外链 → 发外链
新流程：找词（+收集外链）→ 上站 → 筛选高质量外链 → 发外链

**具体操作**：
1. Google 搜索任意关键词，看见搜索结果前3的网站域名是近5年注册的，就把该域名记录下来；竞争激烈的词则记录前两页近5年注册的域名
2. 用 AITDK 查域名注册时间（方便及时看到是否近5年）
3. 用哥飞分享的 Bookmarklet 跳转 ahrefs 外链查询页：`javascript:(function(){var d=window.location.hostname.replace(/^www\./,'');window.open('https://ahrefs.com/backlink-checker/?input='+d+'&mode=subdomains','_blank')})()`
4. 时间积累后，按 DR 由高到低筛查，快速找到高质量外链

**为什么 ahrefs 免费工具有效**：返回的是该网站的 TopBacklinks，即质量最高的那批外链，不是全量外链。实测 DR95 的 dofollow 外链是通过月访问量39.5K、域名2021年注册的网站获得的——所以收集外链时不要关心流量大小，只看域名是否近5年注册且在 Google 搜索结果靠前。

**自动化方案**（进阶）：用 mitmproxy 拦截 ahrefs 的免费 API 请求（`stGetFreeBacklinksOverview`、`stGetFreeBacklinksList`），自动把外链 JSON 数据保存到本地，配合 ZeroOmega 做请求分流，日常使用完全不受影响。评论区有人说直接让 AI 写浏览器插件拦截请求保存也行。评论区还补充了 Cloudflare 人机验证的解决方法（点击高级→键盘调英文→输入"thisisunsafe"回车），以及 macOS 安装证书的方法。

**Google SEO 工具选择**（哥飞）：Semrush 适合初学者（淘宝共享账号十几块/月）；Similarweb 适合赚点钱后（淘宝共享账号70块/月）；Ahrefs 风控严必须买正版（lite 29美元/月，入门版249美元/月），但 Ahrefs 免费版可以验证自己的网站看外链数据，建议每个站长都用。

**关键词密度与排名因果关系**（哥飞 2025-12-29）：不是关键词密度决定排名，而是网页权重决定你能拿到哪些词。正确路径是：选关键词A → 做覆盖AB、AC、AD的页面 → 低权重时先拿AB/AC排名 → 外链+内页增加 → 逐步拿到A的排名。另一个技巧：如果不确定当前权重能拿哪些词，先全面铺，观测到哪个词出排名后，再修改页面突出那个词，其他词的排名也会跟着来。

**Bing vs Google SEO 区别**（哥飞 2025-09-08）：Bing 爬虫弱，依赖站长提交 Sitemap；算法更"传统"，更看重关键词密度、H标签、内链结构；因此做好 On Page SEO 的网站在 Bing 更容易拿排名，但 Bing 算法喜怒无常，有群友新站被 Bing K 站。

**关键词验证陷阱案例（ice breaker games，2026-04-02）**

背景：调研"ice breaker games"（KD 44，shortage 123），初看竞争不激烈、博客占位、有机会。Google Trends 显示 +Breakout 增长。

实查 SERP 后发现：
- 排名第1：Reddit r/Training（月访问38.3亿），全是 B2B 培训内容
- 排名靠前：Figma resource library（8332万）、Miro（3004万）、Mentimeter（439万）
- AI Overview 给的内容：Two Truths and a Lie、10 Things in Common——全是**会议引导技巧**，不是视频游戏
- 底部相关搜索：ice breaker games for work、virtual ice breaker questions——全是 B2B 培训场景

**结论**："ice breaker games"虽然含"games"词，但 Google 把这个词的意图判定为 B2B 会议/团队活动，"games"在这里的意思是"活动"而非"视频游戏"。**意图不在游戏这边，做游戏站永远拿不到这类词的排名。**

这条教训的核心（袁锐钦验证四步法第3步必须实地做）：
> 选关键词不能只看 KD 和搜索量，必须先打开 Google SERP 实际看排名的都是什么类型的页面。
>
> **判断意图的正确顺序**：1. 先看 SERP 排名的页面类型（工具？博客？App Store？论坛？）2. 从页面类型推断 Google 认为用户想要什么 3. 再问自己：我的网站能更好地满足这个意图吗？ 4. 最后才看 KD 和搜索量
>
> **反例教训**：看到"ice breaker games"有 KD 44（不高）+ Breakout（增长）就以为有机会，但 SERP 一查全是 B2B 培训内容，游戏意图为零。KD 再低、趋势再好，意图不对就等于 0。

**另一个验证点**：带"game"词根不等于游戏意图。必须实地看 SERP 里排名的页面是什么——"brainrot game"有游戏竞品（NYT、CrazyGames），"ice breaker games"全是培训内容。

**pips game 教训（2026-04-02）**

KD 23，看似竞争很弱。但 SERP 第一位就是 NYT Games（月访问 6.12 亿），品牌词直接霸屏。仿站 pips-game.com 和 pips-game.org 月访问只有 81K 和 3.5K，说明这个词的流量全被 NYT 吃掉了。

**核心教训**：KD 低 + SERP 有品牌巨头 = 不是机会。KD 低只说明"已排名页面的外链少"，不说明"没有巨头占位"。**必须把 KD 和 SERP 页面类型一起看**：如果第一名是 NYT、App Store、Steam 这类巨头，即使 KD 很低也很难做。

正确做法：KD 低 → 立刻去 SERP 看是谁排第一 → 如果是 NYT/Steam/Amazon 这类品牌站 → 立即放弃。
